Sausage Pasta

Description

This is an expertly crafted sausage meal created by yours truly in university because I eat the same thing 3 days a week

This meal contains 3 simple ingredients, maybe four if you are feeling spicy. Conchigle pasta, prefferably sainsburys own, Lloyd grossmans delicious sauce, and sainsburys taste the difference sausages. Cheese may also be used to add flavour along with salt and pepper

Ingredients

  • Sainsbury's Taste The Difference Caramelised Onion Sausages
  • Sainsbury's Conchigle Pasta
  • Lloyd Grossman Garlic and Basil Pasta Sauce
  • Optional Cheese Makes This Meal Fancy

Steps

  1. Boil some pasta
  2. Cook the sausages
  3. Chop the sausages and put these ingredients together once pasta is cooked and strained
  4. Put the Lloyd Grossman sauce all over this badboy, grate some cheese over it if you feel like it and enjoy!
