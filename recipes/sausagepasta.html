Sausage Pasta

Description

This is an expertly crafted sausage meal created by yours truly in university because I eat the same thing 3 days a week

This meal contains 3 simple ingredients, maybe four if you are feeling spicy. Conchigle pasta, prefferably sainsburys own, Lloyd grossmans delicious sauce, and sainsburys taste the difference sausages. Cheese may also be used to add flavour along with salt and pepper
